# Career Compass

A modern career guidance website helping students explore different career paths, courses, and skills.

## Features
- Interactive career path exploration
- Detailed engineering guide
- Course information and skill requirements
- Mobile responsive design
- Career guidance tips
- Contact form

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Responsive Design
