Engineering Career Guide

Path after 10th Standard

  • Complete 11th & 12th with PCM (Physics, Chemistry, Mathematics)
  • Focus on NCERT books thoroughly
  • Start JEE/CET preparation early
  • Join coaching if needed
  • Practice previous year papers

Entrance Exams

  • JEE Main: National level exam for NITs, IIITs
  • JEE Advanced: For IITs (top 2.5 lakh JEE Main qualifiers)
  • State CET: For state engineering colleges
  • BITSAT: For BITS Pilani campuses
  • VITEEE: For VIT University

Exam Preparation Tips

JEE Main & Advanced

Start preparation in 11th standard. Focus on NCERT books first, then move to advanced references. Practice daily problems and take mock tests regularly.

State CET

Focus on state board syllabus along with CET specific topics. Previous year papers are crucial. Time management is key during the exam.

Study Resources

NCERT Books, Reference books (HC Verma, RD Sharma), Online platforms (Khan Academy, Unacademy), Previous year papers, Mock tests
